# Nadie

Простой статический сайт-приглашение на свидание для Надие.

## GitHub Pages

Сайт состоит из обычных файлов `index.html`, `styles.css` и `script.js`, поэтому его можно
публиковать через GitHub Pages из корня ветки:

1. Откройте настройки репозитория на GitHub.
2. Перейдите в **Pages**.
3. В разделе **Build and deployment** выберите источник **Deploy from a branch**.
4. Укажите нужную ветку и папку `/root`.
5. Откройте опубликованную ссылку GitHub Pages.
